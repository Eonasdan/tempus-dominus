No matter how you choose to get the files, make sure that Popper is include before the picker's main script file.

You will also want a font library. The picker defaults to Font Awesome 6, but you can provide a different icon set via the configuration or a plugin.

Via CDN

    
<!-- Popperjs -->
<script src="https://cdn.jsdelivr.net/npm/@popperjs/core@2.11.2/dist/umd/popper.min.js"
      crossorigin="anonymous"></script>
<!-- Tempus Dominus JavaScript -->
<script src="https://cdn.jsdelivr.net/gh/Eonasdan/tempus-dominus@master/dist/js/tempus-dominus.js"
      crossorigin="anonymous"></script>

<!-- Tempus Dominus Styles -->
<link href="https://cdn.jsdelivr.net/gh/Eonasdan/tempus-dominus@master/dist/css/tempus-dominus.css"
      rel="stylesheet" crossorigin="anonymous">

Via NPM

    
npm install @popperjs/core @eonasdan/tempus-dominus
